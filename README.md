# First-repository
Opis naloge Kviz v NetBeansu.
Pri tej nalogi sem v okolju NetBeans izdelal interaktivni kviz z grafičnim uporabniškim vmesnikom. Kviz vsebuje tri različne teme, med katerimi uporabnik na začetku izbere želeno temo. Vsaka tema ima sedem vprašanj, ki se prikazujejo v ločenih oknih 
(JFrame). Vprašanja so različnih tipov, kot so vnos besedila, izbira z radio gumbi in potrditvena polja. Ob zagonu aplikacije se prikaže pozdravno okno z gumbom za vstop v kviz. Po končanem kvizu posamezne teme se izpiše podrobna statistika reševanja. 
Statistika vključuje število pravilnih in napačnih odgovorov ter odstotek uspešnosti. Uporabniku je omogočena vrnitev na začetni zaslon z izbiro tem.
